## Suggested Workflow

```
Goal 1 → Goal 2 → Goal 3 → Goal 4 → Goal 5 → Goal 6 → Goal 7
```

1. **Goal 1** — Ingest data from SQL into ADLS Gen2 as Parquet
2. **Goal 2** — Register data in Unity Catalog for governance
3. **Goal 3** — Query curated data using SQL
4. **Goal 4** — Build Silver & Gold curated tables
5. **Goal 5** — Add an AI agent for natural language data access
6. **Goal 6** — Mirror data to Microsoft Fabric
7. **Goal 7** — Model the data and build a report/dashboard
